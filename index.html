Help Guide

ZeroApp Builder lets you create Android apps at no cost. Here's how to get started:

  1. Create a new project by clicking "New Project" in the navigation bar
  2. Fill in your app details like name and package
  3. Drag components from the left sidebar onto the canvas
  4. Configure each component using the properties panel on the right
  5. Add multiple screens if needed using the screen selector dropdown
  6. When finished, click "Export APK" to generate your Android app file

The app builder works on a drag-and-drop principle:

  • Left sidebar: Component library - drag items onto the canvas
  • Center: Canvas - visual representation of your app
  • Right sidebar: Properties - customize the selected component

Tips for building:

  • Start with containers and layout components
  • Add UI elements like buttons and text
  • Use the navigation component to link between screens
  • Preview your app in different device sizes with the preview buttons

After exporting your APK, you'll need to:

  1. Create a Google Developer account ($25 one-time fee)
  2. Sign in to the Google Play Console
  3. Create a new app listing
  4. Fill in app metadata (descriptions, screenshots, icon)
  5. Set up app pricing and distribution (select Free)
  6. Upload the signed APK file from ZeroApp Builder
  7. Submit for review

Google typically reviews apps within a few days before they appear on the Play Store.

Important: Keep your keystore file safe! You'll need the same keystore for future app updates.

What are the limitations?
  • Simple apps with basic functionality only
  • Limited access to advanced device features
  • No custom native code insertion
  • Basic design capabilities
Is it really free?

ZeroApp Builder is completely free to use, with no hidden charges for app creation or APK generation. However, to publish on Google Play Store, you need a Google Developer account ($25 one-time fee).

Can I monetize my app?

Yes, you can include ad components and monetize your app through ad networks or make it a paid app on the Play Store.
